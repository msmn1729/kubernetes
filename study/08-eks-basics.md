# 08. EKS Basics

## 범위

- EKS 개념
- 클러스터와 워커 노드 구조
- 로컬에서 EKS 조작 세팅
- ECR/RDS 기반 배포 연결

## 핵심 개념

- EKS는 AWS가 관리하는 Kubernetes control plane 서비스다.
- EC2 k3s보다 관리형 서비스 구조를 이해해야 한다.
- 클러스터 생성, 노드 그룹, kubeconfig 연결, 비용 정리가 핵심 흐름이다.
