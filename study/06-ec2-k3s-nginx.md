# 06. EC2, k3s, Nginx

## 범위

- EC2에서 k3s 설치
- Nginx Deployment/Service 배포
- NodePort 기반 외부 접근

## 핵심 개념

- k3s는 가볍게 실습하기 좋은 쿠버네티스 배포판이다.
- EC2에서는 보안 그룹과 포트 개방을 같이 봐야 한다.
- 로컬 Docker Desktop과 달리 공인 IP, 보안 그룹, 리눅스 환경을 같이 다룬다.
